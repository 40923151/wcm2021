第四~九週

同步 github 和 heroku

要先依序設定這些

完成後就可以在cmd登入你的heroku

再來就可以同步了

heroku 查詢課表app 建立過程

先到老師的倉儲 git calone nfulist 到自己的倉儲

點進來後修改 wsgi.py

去老師的wcm2021裡找

再貼到這上面

另存 檔名叫 wsgi.py 覆蓋掉原本 nfulist 裡的 wsgi.py

再來要把 proxy 屏蔽掉

原因:在heroku開啟app時是外部的連線所以不需要設定proxy

還要新增這兩的變數

1: semester = request.args.get('semester') 

2:classroomno = request.args.get('classroomno')

沒理解錯誤的話 semester 是控制第幾學年的第幾個學期

                     classroomno 是控制要第幾間教室的課表

都改完就可以把 nfulist 推到 heroku

再來打開 app

改改看變數 有成功換教室就是完成了

中間要用問號隔開

?classroomno=BGA0730
